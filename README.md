# Cision Feedback App

1. Clone repo
2. `npm install`
3. `gulp`

Make sure you have gulp installed globally (`npm install -g gulp`)

##Deploying
1. npm install -g firebase-tools
2. `firebase login`
3. `firebase use default` || `firebase use stage`
